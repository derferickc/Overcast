playlistWithURI:session:callback:

Request the playlist at the given Spotify URI.

+ (void)playlistWithURI:(NSURL *)uri session:(SPTSession *)session callback:(SPTRequestCallback)block

Parameters

uri

The Spotify URI of the playlist to request.

session

An authenticated session. Must be valid and authenticated with the SPTAuthPlaylistReadScope or SPTAuthPlaylistReadPrivateScope scope as necessary.

block

The block to be called when the operation is complete. The block will pass a Spotify SDK metadata object on success, otherwise an error.

Discussion

Note: This method takes Spotify URIs in the form spotify:*, NOT HTTP URLs.
