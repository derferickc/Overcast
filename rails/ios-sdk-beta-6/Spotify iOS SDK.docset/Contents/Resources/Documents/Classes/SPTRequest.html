Performs a search with a given query and offset

+ (void)performSearchWithQuery:(NSString *)searchQuery queryType:(SPTSearchQueryType)searchQueryType offset:(NSInteger)offset session:(SPTSession *)session callback:(SPTRequestCallback)block

Parameters

searchQuery

The query to pass to the search.

searchQueryType

The type of search to do.

offset

The index at which to start returning results.

session

An authenticated session. Can be nil.

block

The block to be called when the operation is complete. The block will pass an SPTListPage containing results on success, otherwise an error.

performSearchWithQuery:queryType:offset:session:market:callback:

Performs a search with a given query, offset and market filtering

+ (void)performSearchWithQuery:(NSString *)searchQuery queryType:(SPTSearchQueryType)searchQueryType offset:(NSInteger)offset session:(SPTSession *)session market:(NSString *)market callback:(SPTRequestCallback)block

Parameters

searchQuery

The query to pass to the search.

searchQueryType

The type of search to do.

offset

The index at which to start returning results.

session

An authenticated session. Can be nil.

market

Either a ISO 3166-1 country code to filter the results to, or from_token to pick the market from the session (requires the user-read-private scope), or nil for no market filtering.

block

The block to be called when the operation is complete. The block will pass an SPTListPage containing results on success, otherwise an error.

requestFeaturedPlaylistsForCountry:limit:offset:locale:timestamp:session:callback:

Get a list of featured playlists

+ (void)requestFeaturedPlaylistsForCountry:(NSString *)country limit:(NSInteger)limit offset:(NSInteger)offset locale:(NSString *)locale timestamp:(NSDate *)timestamp session:(SPTSession *)session callback:(SPTRequestCallback)block

Parameters

country

A ISO 3166-1 country code to get playlists for, or nil to get global recommendations.

limit

The number of results to return, max 50.

offset

The index at which to start returning results.

locale

The locale of the user, for localized recommendations, nil will default to American English.

timestamp

The time of day to get recommendations for (without timezone), or nil for current local time

session

An authenticated session. Must be valid and authenticated with the

block

The block to be called when the operation is complete, containing a SPTFeaturedPlaylistList

Discussion

See https://developer.spotify.com/web-api/get-list-featured-playlists/ for more information on parameters
