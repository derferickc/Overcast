albumWithURI:session:callback:

Request the album at the given Spotify URI.

+ (void)albumWithURI:(NSURL *)uri session:(SPTSession *)session callback:(SPTRequestCallback)block

Parameters

uri

The Spotify URI of the album to request.

session

An authenticated session. Can be nil.

block

The block to be called when the operation is complete. The block will pass a Spotify SDK metadata object on success, otherwise an error.

Discussion

Note: This method takes Spotify URIs in the form spotify:*, NOT HTTP URLs.
